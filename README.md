# GPA & CGPA Calculator Android App

An Android application built with Kotlin/Java that allows students to calculate:

- Semester GPA
- Cumulative GPA (CGPA)

## Features
- Add multiple semesters
- Add courses with credit hours and grades
- Automatic GPA calculation per semester
- CGPA calculation across all semesters
- Simple and student-friendly UI

## How It Works
1. Open the app
2. Add a semester
3. Enter courses (name, credit hour, grade)
4. App calculates GPA automatically
5. CGPA is updated based on all semesters

## Tech Stack
- Kotlin / Java
- Android Studio
- XML Layouts
- Android SDK

This project was built to help students easily track their academic performance and to improve my Android development skills.
